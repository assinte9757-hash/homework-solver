# 作业解题助手 📚

一个基于网页的作业拍照解题小工具，支持OCR识别题目、AI智能解答、答案纠错、历史记录和PDF导出功能。

## 功能特性

- 📷 **拍照识别**：直接用手机相机拍照，或从相册选择作业图片
- 🔍 **OCR识别**：自动识别图片中的作业内容（支持阿里云、百度、腾讯云、OpenAI）
- 🧠 **AI解题**：使用阿里云通义千问等AI模型进行智能解题
- 📝 **详细答案**：提供解题思路、完整答案和考察知识点
- ✅ **答案纠错**：对于自己写的答案，自动识别错误并高亮标注
- 💾 **历史记录**：自动保存识别历史，随时查看
- 📄 **PDF导出**：支持将答案导出为PDF文件，方便打印保存
- 🎨 **美观界面**：现代化的UI设计，适配手机和电脑

## 快速开始

### 1. 部署方式

这是一个纯前端静态网页应用，你可以通过以下方式部署：

- 直接双击 `index.html` 在浏览器打开
- 部署到任意静态网站托管服务（如GitHub Pages、Vercel、Netlify等）
- 部署到自己的服务器
- 通过局域网访问

### 2. 配置API密钥

首次使用需要配置AI服务的API密钥：

1. 点击右上角的「设置」按钮
2. 选择AI服务提供商（默认：阿里云）
3. 填写对应的API密钥：
   - **阿里云**：需要AccessKey ID和AccessKey Secret
   - **百度智能云**：需要API Key和Secret Key
   - **腾讯云**：需要SecretId和SecretKey
   - **OpenAI**：需要API Key（可选自定义Base URL）

### 3. 申请API密钥

#### 阿里云（推荐）
1. 访问 [阿里云官网](https://www.aliyun.com) 注册账号
2. 开通「通义千问」和「文字识别OCR」服务
3. 创建AccessKey：控制台 > AccessKey管理
4. 阿里云新用户有免费额度，足够日常使用

#### 百度智能云
1. 访问 [百度智能云](https://cloud.baidu.com) 注册账号
2. 开通「文字识别OCR」和「千帆大模型」服务
3. 创建应用获取API Key和Secret Key

#### 腾讯云
1. 访问 [腾讯云](https://cloud.tencent.com) 注册账号
2. 开通「OCR」和「混元大模型」服务
3. 获取SecretId和SecretKey

#### OpenAI
1. 访问 [OpenAI官网](https://openai.com) 注册账号
2. 创建API Key
3. 需要使用代理才能访问

## 使用说明

### 基本使用流程

1. **拍照/选择图片**
   - 点击「拍照」按钮使用相机拍摄作业
   - 点击「相册」从手机相册选择图片

2. **识别作业**
   - 拍照后点击「开始识别」
   - 等待OCR识别和AI解题完成

3. **查看答案**
   - 查看每个题目的解题思路
   - 查看完整答案
   - 查看考察的知识点

4. **答案纠错**
   - 如果自己写了答案，AI会自动检查
   - 错误部分会以红色高亮显示
   - 正确答案会以绿色显示

5. **导出/保存**
   - 点击「导出PDF」保存答案
   - 系统会自动保存历史记录
   - 可以随时查看历史记录

### 功能详解

#### 历史记录
- 点击右上角的「历史记录」按钮
- 查看所有之前的识别记录
- 可以查看、删除单条记录
- 记录保存在浏览器本地，不会上传到服务器

#### PDF导出
- 在答案页面点击「导出PDF」按钮
- 自动生成包含所有题目答案的PDF文件
- 可以打印或保存

#### 答案纠错
- 在设置中启用「答案纠错功能」
- 对于选择题、判断题，AI会自动对比你的答案
- 错误答案会被标注出来

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript（无需构建工具）
- **OCR服务**：阿里云OCR、百度OCR、腾讯云OCR、OpenAI GPT-4 Vision
- **AI模型**：阿里云通义千问、百度文心一言、腾讯混元、OpenAI GPT
- **UI库**：Font Awesome（图标）、html2canvas、jsPDF
- **数据存储**：localStorage（本地存储）

## 注意事项

1. **API密钥安全**：API密钥只保存在浏览器本地，不会上传到任何服务器
2. **网络环境**：需要联网才能使用OCR和AI服务
3. **图片质量**：拍照时请确保光线充足、图片清晰、题目完整
4. **费用说明**：各大云服务都有免费额度，超出后按量计费
5. **浏览器兼容**：推荐使用Chrome、Safari、Edge等现代浏览器
6. **HTTPS要求**：使用相机功能需要HTTPS环境（本地localhost除外）

## 隐私说明

- 所有数据仅保存在用户浏览器本地
- 不收集任何个人信息
- API密钥仅用于调用云服务，不会上传到任何第三方
- 历史记录仅在本地存储

## 常见问题

### Q: 无法访问相机怎么办？
A: 请确保：
1. 浏览器有相机权限
2. 使用HTTPS或localhost访问
3. 检查手机系统设置中是否禁止了相机权限

### Q: OCR识别不准确怎么办？
A: 建议：
1. 确保拍摄时光线充足
2. 尽量垂直拍摄，避免倾斜
3. 确保题目完整在画面内
4. 尝试切换到其他OCR服务

### Q: AI生成的答案不正确怎么办？
A: AI仅供参考学习使用，建议：
1. 自己先思考解题思路
2. 对比AI的答案进行验证
3. 有疑问可以向老师或同学请教

### Q: API密钥如何保管？
A: 建议：
1. 不要将API密钥分享给他人
2. 定期更换API密钥
3. 使用完工具后可以清除浏览器缓存
